# 개요

## 특징
- (Web Real-Time Communication)는 웹 브라우저와 모바일 앱 사이에 실시간 통신을 위한 기술이다. 
- 브라우저 간에 오디오, 비디오 및 데이터를 교환할 수 있도록 설계되었다.
- 브라우저 간에 피어 투 피어(P2P) 통신을 가능하게 하며, 중간 서버가 필요하지 않다. 
- 브라우저에서 사용할 수 있는 API를 제공하며, 이를 통해 오디오, 비디오 및 데이터를 교환할 수 있다.
