# Istio Service Mesh
서비스 네트워크를 제공하는 오픈 소스 서비스 메시 플랫폼입니다. Istio는 서비스 간 통신, 보안, 통제 및 모니터링을 제공합니다.
자동화 된 프레임워크 스타일의 플러그인은 지향하고 싶었지만.. 결국 돌고돌아 Istio를 사용하게 되었다.  

## Getting Started 
```bash
$ curl -L https://istio.io/downloadIstio | sh -
Istio 1.21.2 Download Complete!
Istio has been successfully downloaded into the istio-1.21.2 folder on your system.
Next Steps:
See https://istio.io/latest/docs/setup/install/ to add Istio to your Kubernetes cluster.
To configure the istioctl client tool for your workstation,
add the /root/istio-1.21.2/bin directory to your environment path variable with:
         export PATH="$PATH:/root/istio-1.21.2/bin"
```

친절하게 환경변수 커맨드를 주니 그대로 붙여넣자
```bash
$ tee -a ~/.bashrc <<EOF
export PATH="$PATH:/root/istio-1.21.2/bin"
EOF

$ istioctl x precheck 
✔ No issues found when checking the cluster. Istio is safe to install or upgrade!
```
이슈 없음 확인, 넘어가자  
다음 설치에는 `demo` 프로필을 사용합니다. 테스트를 위한 적절한 기본값 세트를 선택했지만 다른 프로필(프로덕션 또는 성능 테스트, AWS, GCP등)도 고려하세요
```bash
$ istioctl install --set profile=demo -y
