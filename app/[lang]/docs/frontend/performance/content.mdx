# 개요
성능 최적화는 사용자 경험을 향상시키고, 사용자 유지율을 높이며, 비즈니스 성과를 향상시킵니다.   
기본적인 최적화 요소들에대해 알아보고, Vue.js관점에서 성능 최적화를 위한 방법을 알아보겠습니다.  


## 브라우저 렌더링 절차
최적화를 위해서는 브라우저 렌더링 절차를 이해해야 합니다.  
브라우저 렌더링 절차는 다음과 같습니다.
1. HTML 파일을 파싱해서 DOM 트리를 생성합니다.
   1. 이과정에서 JS 파일을 만나면 파싱 및 Dom 생성을 중단하고 JS 파일을 다운로드하고 실행합니다.  
   2. 이때 재 렌더링이 발생 할 수 있습니다.
2. CSS 파일을 파싱해서 CSSOM 트리를 생성합니다.
3. DOM 트리와 CSSOM 트리를 결합해서 렌더 트리를 생성합니다.
4. 렌더 트리를 기반으로 레이아웃을 계산합니다.
5. 레이아웃을 기반으로 픽셀(Painting)을 그립니다. 
6. JavaScript 파일을 파싱하고 실행합니다.
7. 렌더링이 완료되면 화면에 표시합니다.
8. 사용자 입력을 받아서 반응합니다.
9. 사용자 입력에 따라 렌더링을 다시 수행합니다.
10. 사용자 입력에 따라 JavaScript를 실행합니다.
11. 사용자 입력에 따라 화면을 갱신합니다.
12. 사용자 입력에 따라 서버로 요청을 보냅니다.
13. 서버로부터 응답을 받아서 화면을 갱신합니다.
14. 서버로부터 응답을 받아서 JavaScript를 실행합니다.
15. 서버로부터 응답을 받아서 화면을 갱신합니다.
