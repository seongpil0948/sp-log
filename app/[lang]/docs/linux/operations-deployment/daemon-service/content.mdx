# Daemon
데몬은 리눅스 시스템에서 백그라운드에서 실행되는 프로그램입니다. 
사용자와 직접 상호 작용하지 않고 특정 작업을 수행하도록 설계되었습니다.  
데몬은 일반적으로 시스템 부팅 시 자동으로 시작되며 시스템 종료될 때까지 계속 실행됩니다.

## 데몬의 주요 특징

- **백그라운드 실행**: 데몬은 사용자 인터페이스 없이 백그라운드에서 실행됩니다.
- **지속적인 실행**: 데몬은 시스템 부팅 시 자동으로 시작되며 시스템 종료될 때까지 계속 실행됩니다.
- **자동화된 작업**: 데몬은 특정 작업을 자동화하도록 설계되었습니다.
- **서비스 제공**: 데몬은 네트워킹, 프린팅, 파일 시스템 관리 등 다양한 서비스를 제공합니다.
- 데몬의 종류:
  - **독립형 데몬**: 독립적으로 실행되는 데몬입니다.
  - **슈퍼 데몬(inetd)**: inetd 데몬에 의해 관리되는 데몬입니다.
## 데몬의 예시
- **sshd**: SSH 서버 데몬
- **httpd**: 웹 서버 데몬
- **mysqld**: MySQL 데이터베이스 서버 데몬
- **crond**: cron 작업 관리 데몬
- **syslogd**: 시스템 로그 관리 데몬

## 데몬 관리
- **시스템ctl**: systemd를 사용하여 데몬을 시작, 중지, 재시작할 수 있습니다.
- **init**: init 스크립트를 사용하여 데몬을 시작, 중지, 재시작할 수 있습니다.
- **서비스 관리 도구**: chkconfig, upstart와 같은 서비스 관리 도구를 사용하여 데몬을 관리할 수 있습니다.

## 데몬에 대한 추가 정보:
- 데몬은 일반적으로 d로 끝나는 이름을 가지고 있습니다. 예: sshd, httpd
- 데몬은 /etc/init.d 또는 /etc/systemd/system 디렉토리에 있습니다.
- 데몬의 구성 파일은 /etc 디렉토리에 있습니다.
- 데몬은 리눅스 시스템에서 중요한 역할을 합니다. 데몬은 시스템 유지 관리, 서비스 제공, 자동화된 작업 수행 등 다양한 기능을 제공합니다. 데몬에 대한 이해는 리눅스 시스템을 관리하는 데 필수적입니다.
