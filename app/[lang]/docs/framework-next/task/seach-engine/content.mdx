# 소개
작성한 문서들이 많아지다보니 실무에 바로바로 검색할 수 있도록 검색 엔진을 구현하자 결정했습니다.  

### 제한사항
- 저는 돈이 없습니다. Elasticsearch나 Algolia 같은 서비스를 사용할 수 없습니다.  
- MDX, MD 파일을 검색할 수 있어야 합니다.  
- 검색 엔진은 서버사이드에서 동작해야 합니다.
- 불변데이터로 실시간 검색이 필요하지 않습니다. 
캐싱을 최대한 활용하여 검색 속도를 높이는 것이 목표입니다.  

### 사용할 기술
- Node.js
- Firebase Firestore  
- MDX, MD 파일을 읽을 수 있는 라이브러리
